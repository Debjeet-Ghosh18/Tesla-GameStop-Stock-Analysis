# Analyzing Historical Stock Revenue Data and Building a Dashboard

## Project Overview
This project analyzes historical stock prices and revenue data for Tesla (TSLA) and GameStop (GME) using Python.

The project uses:
- yfinance
- BeautifulSoup
- requests
- pandas
- plotly

The dashboard visualizes:
- Historical stock prices
- Historical company revenue

---

## Technologies Used

- Python
- Jupyter Notebook
- yfinance
- BeautifulSoup4
- pandas
- requests
- plotly
